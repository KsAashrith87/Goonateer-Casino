HOW TO RUN GOONATEER CASINO

1. Unzip the folder.
2. Open Terminal.
3. Type: cd ~/Downloads/goonateer_casino
   If your folder has a different name, drag the folder into Terminal after typing cd and a space.
4. Type: npm install
5. Type: npm start
6. Open: http://localhost:3000

Mac shortcut:
Right-click RUN_GOONATEER.command, click Open, then Open again.
Keep the Terminal window open while playing.
